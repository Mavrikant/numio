# 시계열 시각화 도구

날짜 열이 포함된 CSV 파일을 업로드하고, 하나 이상의 수치 값 열을 선택하면 브라우저에서 즉시 인터랙티브 꺾은선 그래프를 생성할 수 있습니다. 중심화 이동 평균을 추가하여 노이즈를 줄이고 기저 추세를 드러낼 수 있습니다.

## 시계열이란?

**시계열**은 시간 순서로 정렬된 데이터 포인트의 시퀀스입니다. 일별 주가, 시간별 온도, 주별 웹사이트 트래픽 등이 그 예입니다. 시간은 항상 x축에, 하나 이상의 수치량은 y축에 위치합니다.

시계열을 시각화하면 다음과 같은 질문에 답할 수 있습니다:

- 분석 기간 동안 상승 또는 하락 **추세**가 있는가?
- 예측 가능한 간격으로 반복되는 **계절적 패턴** — 정점과 저점이 있는가?
- 외부 이벤트를 암시하는 갑작스러운 **급등 또는 급락**이 있는가?
- 전반적인 추세에 비해 **변동성**(단기 변동)은 어느 정도인가?

## 이동 평균

**단순 이동 평균(SMA)**은 각 값을 주변 윈도우 값들의 평균으로 대체하여 시리즈를 부드럽게 만듭니다. 이 도구는 **중심화** SMA를 사용합니다: 윈도우 크기 _w_의 경우, 인덱스 _i_에서의 평활화된 값은 인덱스 _i − ⌊w/2⌋_부터 _i + ⌊w/2⌋_까지의 평균입니다. 완전한 윈도우를 갖지 않는 가장자리 포인트는 0으로 채우지 않고 제외됩니다.

**윈도우 크기 선택 방법:**

- **소(3–5):** 가벼운 평활화로 세부 사항 유지——일별 데이터에 적합.
- **중(7–14):** 일별 데이터의 1~2주 평균; 주식 차트의 고전적 선택.
- **대(30–90):** 장기 추세를 드러내고 계절 변동을 억제.

## 사용 방법

1. CSV 업로드(드래그 앤 드롭 또는 붙여넣기).
2. 드롭다운에서 시간 열 선택——날짜 열은 자동으로 감지됩니다.
3. 값 열을 체크박스로 선택; 각각 고유한 색상의 선이 생성됩니다.
4. 필요하면 이동 평균을 활성화하고 윈도우 크기를 설정.
5. 차트 아래 요약 테이블 확인: 최소, 최대, 평균, 최신 값, 및 총 변화율.

## 일반적인 활용 사례

- **금융:** 여러 자산의 일별 종가를 나란히 비교.
- **기상:** 수개월 또는 수년에 걸친 온도 또는 강수량 시각화.
- **웹 분석:** 일별 또는 주별 페이지뷰, 세션, 또는 전환율 추적.
- **과학:** 실험 단계나 시간 간격에 걸친 측정값 표시.

## 개인 정보 보호

모든 계산은 브라우저에서 로컬로 실행됩니다. 서버로 데이터가 전송되지 않습니다. 최대 파일 크기: 5 MB.
