# Визуализатор Временных Рядов

Загрузите CSV-файл со столбцом дат, выберите один или несколько числовых столбцов значений и мгновенно получите интерактивный линейный график — прямо в браузере. Добавьте центрированное скользящее среднее для сглаживания шума и выявления трендов.

## Что такое временной ряд?

**Временной ряд** — это последовательность данных, упорядоченных по времени: ежедневные котировки акций, почасовая температура, еженедельный трафик сайта. Время всегда откладывается по оси X, числовые величины — по оси Y.

Визуализация временного ряда помогает ответить на вопросы:

- Есть ли восходящий или нисходящий **тренд** за рассматриваемый период?
- Присутствуют ли **сезонные паттерны** — регулярные пики и впадины?
- Есть ли резкие **скачки или провалы**, свидетельствующие о внешних событиях?
- Какова **волатильность** (краткосрочные колебания) относительно общего тренда?

## Скользящее среднее

**Простое скользящее среднее (СС)** сглаживает ряд, заменяя каждое значение средним по окружающему окну. Инструмент использует **центрированное** СС: при ширине окна _w_ сглаженное значение в позиции _i_ равно среднему по индексам от _i − ⌊w/2⌋_ до _i + ⌊w/2⌋_. Крайние точки, для которых не хватает соседей, исключаются, а не заполняются нулями.

**Выбор размера окна:**

- **Малое (3–5):** лёгкое сглаживание, сохраняет детали — подходит для дневных данных.
- **Среднее (7–14):** одна-две недели для дневных данных; классический выбор для биржевых графиков.
- **Большое (30–90):** выявляет долгосрочный тренд, подавляет сезонные колебания.

## Как пользоваться

1. Загрузить CSV (перетащить или вставить).
2. Выбрать столбец времени из выпадающего списка — столбцы с датами определяются автоматически.
3. Отметить столбцы значений галочками; каждый становится отдельной цветной линией.
4. При необходимости включить скользящее среднее и задать размер окна.
5. Прочитать таблицу сводной статистики: мин, макс, среднее, последнее значение и изменение в процентах.

## Типичные сценарии использования

- **Финансы:** сравнивать дневные цены закрытия нескольких активов.
- **Погода и климат:** визуализировать температуру или осадки за месяцы и годы.
- **Веб-аналитика:** отслеживать просмотры страниц, сессии или конверсии по дням или неделям.
- **Наука:** отображать измерения по шагам эксперимента.

## Конфиденциальность

Все вычисления выполняются локально в браузере. Никакие данные не передаются на сервер. Максимальный размер файла: 5 МБ.
