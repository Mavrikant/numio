## 简介

梁挠度计算是结构工程中最基础的计算之一。当梁承受荷载时会发生弯曲，必须对这种变形进行控制，以确保结构完整性和使用者舒适度。过大的挠度可能导致饰面开裂、设备失准，以及屋面积水。

本计算器采用**欧拉-伯努利梁理论**，计算简支梁和悬臂梁在三种常见荷载工况下的最大挠度（mm）、最大弯矩（N·m）、最大剪力（N）及支座反力（N）。

## 梁的类型

**简支梁：** 两端分别设置铰支座和滑动支座，是钢结构和木结构楼板梁最常见的形式。

**悬臂梁：** 一端完全固定，另一端自由。常见于阳台、屋檐及飞机机翼结构。

## 荷载类型

- 跨中集中荷载
- 自由端集中荷载（仅适用于悬臂梁）
- 均布荷载

## 挠度公式

由欧拉-伯努利微分方程 EI(d²y/dx²) = M 推导而来：

| 工况 | 公式 |
|---|---|
| 简支梁，跨中集中荷载 | δ = FL³ / (48EI) |
| 简支梁，均布荷载 | δ = 5wL⁴ / (384EI) |
| 悬臂梁，自由端集中荷载 | δ = FL³ / (3EI) |
| 悬臂梁，均布荷载 | δ = wL⁴ / (8EI) |

## 弹性模量

| 材料 | E（GPa） |
|---|---|
| 结构钢 | 200–210 |
| 铝合金 | 68–72 |
| 钢筋混凝土 | 25–35 |
| 软木（顺纹） | 8–12 |
| 硬木 | 10–20 |

## 截面惯性矩

截面惯性矩 I 衡量截面抵抗弯曲的能力。截面越深，刚度越大，因为 I 与截面高度的三次方成正比。

**矩形截面：** I = (b × h³) / 12

**圆形截面：** I = π × d⁴ / 64

标准钢型材（工字钢、H型钢）的 I 值可查阅型钢规格表。

## 变形限值

结构设计规范规定了在可变荷载作用下的挠度限值：

| 构件 | 限值 |
|---|---|
| 楼面梁 | L/300 至 L/360 |
| 屋面梁（无抹灰） | L/200 |
| 屋面梁（有抹灰） | L/360 |
| 悬臂梁 | L/180 |

## 弯矩与剪力

计算器还输出最大弯矩和最大剪力，用于**强度验算**：

- **弯矩**决定所需截面抵抗矩
- **剪力**控制腹板抗剪验算

支座反力是设计支承结构的关键输入。

## 假设条件与局限

本计算器基于欧拉-伯努利梁理论假设：

1. 线弹性材料（无屈服）
2. 小变形（δ &lt;&lt; L）
3. 均质等截面梁
4. 忽略剪切变形
5. 无轴向荷载

**所有结构设计必须由注册结构工程师验证。**

## 常见问题

**为什么悬臂梁的挠度远大于简支梁？**

在相同荷载和跨度下，悬臂梁的挠度是简支梁的16倍（FL³/3EI对比FL³/48EI）。这一差距源于不同的边界条件：悬臂梁自由端无转动约束，也没有第二个支座的刚度贡献。

**什么是抗弯刚度EI？**

EI是弹性模量（E）与截面惯性矩（I）的乘积，是所有挠度公式中刚度的唯一量度。要将挠度减半，必须将EI加倍。
