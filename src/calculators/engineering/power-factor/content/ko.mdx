역률 계산기는 전기 시스템에서 유효 전력, 피상 전력, 무효 전력 사이의 관계를 분석합니다. 세 가지 동작 모드를 제공합니다: 전력 삼각형에서 역률을 계산하는 모드, 역률에서 전력 성분을 계산하는 모드, 그리고 커패시터를 사용한 역률 개선 모드입니다.

## 역률이란?

역률(PF)은 유효 전력(P)과 피상 전력(S)의 비율입니다. 0에서 1 사이의 무차원 수입니다. 높은 역률은 전기 에너지의 효율적인 사용을 나타내고, 낮은 역률은 불필요한 무효 전력 순환을 의미합니다.

**전력 삼각형 성분:**

- **P(유효 전력, kW):** 실제 일을 하는 전력 — 조명, 모터 구동.
- **Q(무효 전력, kVAR):** 전기장과 자기장에 저장·방출되는 전력. 모터와 변압기에 존재합니다.
- **S(피상 전력, kVA):** 유효 전력과 무효 전력의 벡터 합: S = √(P² + Q²)

역률: **PF = cos(φ) = P / S**

위상각: **φ = arccos(PF)**

## 역률의 중요성

**경제적 영향:** 역률이 낮으면 산업 시설은 무효 전력 요금을 납부해야 합니다. 무효 전력은 유용한 일을 하지 않으면서 전선과 변압기에 부하를 줍니다.

**기술적 영향:**
- 낮은 PF는 더 큰 단면적의 도선이 필요합니다
- 변압기와 발전기의 유효 용량이 감소합니다
- 시스템 손실이 증가합니다(I²R 손실)
- 전압 강하가 커집니다

**표준:** IEEE와 IEC는 일반적으로 산업 설비에 대해 PF ≥ 0.90을 권장합니다.

## 동작 모드

### 모드 1: 전력에서 역률 계산

유효 전력(kW)과 무효 전력(kVAR)에서 역률을 계산합니다.

**공식:**
- S = √(P² + Q²) kVA
- PF = P / S
- φ = arccos(PF) 도

이 모드는 전력 분석기나 전력 기록계로 측정한 데이터에 적합합니다.

### 모드 2: 역률에서 전력 성분 계산

피상 전력(kVA)과 역률에서 유효 전력과 무효 전력 성분을 계산합니다.

**공식:**
- P = S × PF kW
- Q = √(S² - P²) kVAR

변압기 및 발전기 용량 계획에 유용합니다.

### 모드 3: 역률 개선

역률을 목표값으로 개선하는 데 필요한 커패시터 용량을 계산합니다.

**공식:**
- Q_cap = P × (tan φ₁ - tan φ₂) kVAR
- C = Q_cap × 1000 / (2π × f × V²) 패럿
- C_µF = C × 10⁶ 마이크로패럿

여기서:
- φ₁ = arccos(현재 PF)
- φ₂ = arccos(목표 PF)
- f = 계통 주파수(50 또는 60 Hz)
- V = 선간 전압(볼트)

## 커패시터를 이용한 역률 개선

유도성 부하(모터, 안정기, 변압기)는 전류가 전압보다 지연되어 지연 역률을 발생시킵니다. 병렬 커패시터는 무효 전력을 국소적으로 공급하여 다음을 가능하게 합니다:

1. 계통에서 끌어오는 무효 전류 감소
2. 피상 전력 수요 감소
3. 도선의 손실 감소
4. 전압 프로파일 개선

**주의:** 이 계산은 기본 주파수에서의 선형 부하에 대해 유효합니다. 고조파를 발생시키는 부하(인버터, UPS, 전기 아크로)는 전문적인 고조파 분석이 필요합니다.

## 실제 계산 예시

**시나리오:** 75 kW 모터가 역률 0.70으로 동작하고 있습니다. 목표: PF를 0.95로 개선.

**1단계:** 현재 상태 계산
- φ₁ = arccos(0.70) = 45.57°
- Q₁ = 75 × tan(45.57°) = 76.5 kVAR
- S₁ = 75 / 0.70 = 107.1 kVA

**2단계:** 목표 상태 계산
- φ₂ = arccos(0.95) = 18.19°
- Q₂ = 75 × tan(18.19°) = 24.7 kVAR

**3단계:** 커패시터 용량 계산
- Q_cap = 75 × (1.020 - 0.329) = 51.8 kVAR
- 400 V, 50 Hz에서: C = 51 800 / (2π × 50 × 400²) = 1 030 µF

**결과:** 피상 전력이 107.1 kVA에서 78.9 kVA로 감소하여 손실이 크게 줄어듭니다.

## 역률 평가표

| 역률 | 평가 | 조치 |
|---|---|---|
| 0.95 – 1.00 | 우수 | 개선 불필요 |
| 0.90 – 0.95 | 양호 | 모니터링 |
| 0.80 – 0.90 | 보통 | 개선 권장 |
| 0.70 – 0.80 | 낮음 | 커패시터 필요 |
| &lt; 0.70 | 매우 낮음 | 즉시 개선 |
