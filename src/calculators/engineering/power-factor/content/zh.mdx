# 功率因数计算器

功率因数计算器分析电气系统中有功功率、视在功率和无功功率之间的关系。提供三种工作模式：从功率三角形计算功率因数、从功率因数计算功率分量，以及通过电容器进行功率因数补偿。

## 什么是功率因数？

功率因数（PF）是有功功率（P）与视在功率（S）之比，是0到1之间的无量纲数。高功率因数表示电能利用效率高；低功率因数表示存在不必要的无功功率循环。

**功率三角形分量：**

- **P（有功功率，kW）：** 实际做功的功率——照明、驱动电机。
- **Q（无功功率，kVAR）：** 在电场和磁场中储存和释放的功率，存在于电机和变压器中。
- **S（视在功率，kVA）：** 有功和无功的向量和：S = √(P² + Q²)

功率因数：**PF = cos(φ) = P / S**

相位角：**φ = arccos(PF)**

## 功率因数的重要性

**经济影响：** 功率因数低时，工业设施需缴纳无功电费。无功功率占用线路和变压器容量却不做有用功。

**技术影响：**
- 低PF需要更大截面的导线
- 变压器和发电机的有效容量降低
- 系统损耗增加（I²R损耗）
- 电压降加大

**标准：** IEEE和IEC通常建议工业装置PF ≥ 0.90。

## 工作模式

### 模式一：从功率计算功率因数

由有功功率（kW）和无功功率（kVAR）计算功率因数。

**公式：**
- S = √(P² + Q²) kVA
- PF = P / S
- φ = arccos(PF) 度

此模式适合功率分析仪或电能记录仪的测量数据。

### 模式二：从功率因数计算功率分量

由视在功率（kVA）和功率因数计算有功和无功分量。

**公式：**
- P = S × PF kW
- Q = √(S² - P²) kVAR

用于变压器和发电机容量规划。

### 模式三：功率因数补偿

计算将功率因数提升至目标值所需的电容器容量。

**公式：**
- Q_cap = P × (tan φ₁ - tan φ₂) kVAR
- C = Q_cap × 1000 / (2π × f × V²) 法拉
- C_µF = C × 10⁶ 微法

其中：
- φ₁ = arccos(当前PF)
- φ₂ = arccos(目标PF)
- f = 电网频率（50或60 Hz）
- V = 线电压（伏特）

## 电容器补偿原理

感性负载（电机、镇流器、变压器）因电流滞后电压而产生滞后功率因数。并联电容器就地提供无功功率，可以：

1. 减少从电网吸取的无功电流
2. 降低视在功率需求
3. 减少导线中的损耗
4. 改善电压分布

**注意：** 此计算适用于基波频率下的线性负载。产生谐波的负载（变频器、UPS、电弧炉）需要专业的谐波分析。

## 实际计算示例

**场景：** 75 kW电机以功率因数0.70运行，目标：将PF提升至0.95。

**第一步：** 计算当前状态
- φ₁ = arccos(0.70) = 45.57°
- Q₁ = 75 × tan(45.57°) = 76.5 kVAR
- S₁ = 75 / 0.70 = 107.1 kVA

**第二步：** 计算目标状态
- φ₂ = arccos(0.95) = 18.19°
- Q₂ = 75 × tan(18.19°) = 24.7 kVAR

**第三步：** 计算电容器容量
- Q_cap = 75 × (1.020 - 0.329) = 51.8 kVAR
- 在400 V、50 Hz下：C = 51 800 / (2π × 50 × 400²) = 1 030 µF

**结果：** 视在功率从107.1 kVA降至78.9 kVA；损耗显著降低。

## 功率因数评估表

| 功率因数 | 评级 | 建议 |
|---|---|---|
| 0.95 – 1.00 | 优秀 | 无需补偿 |
| 0.90 – 0.95 | 良好 | 监测 |
| 0.80 – 0.90 | 一般 | 建议补偿 |
| 0.70 – 0.80 | 偏低 | 需要电容器 |
| &lt; 0.70 | 很低 | 立即补偿 |
