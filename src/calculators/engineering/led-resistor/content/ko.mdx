## LED 전류 제한 저항이란?

LED(발광 다이오드)는 전류가 흐를 때 빛을 내는 반도체 부품입니다. 일반 저항이나 백열전구와 달리 LED에는 전류를 스스로 제한하는 메커니즘이 없습니다. 외부 저항이 없으면 공급 전압이 조금만 상승해도 파괴적인 과전류가 발생해 LED가 영구적으로 손상됩니다.

직렬 연결된 전류 제한 저항은 가장 간단하고 신뢰할 수 있는 해결책입니다. 공급 전압과 LED 순방향 전압의 차이를 열로 변환하여 LED 전류를 안전한 수준으로 유지합니다.

기본 공식은 옴의 법칙에서 바로 도출됩니다:

**R = (V공급 − V순방향 × n) / I순방향**

여기서:
- **R** = 필요한 저항값(Ω)
- **V공급** = 직류 공급 전압(예: 5V, 12V)
- **V순방향** = 단일 LED의 순방향 전압(색상에 따라 1.8V–3.6V)
- **n** = 직렬 LED 수
- **I순방향** = 목표 LED 전류(암페어, 20mA = 0.020A)

저항 소비 전력: **P = (V공급 − V순방향 × n) × I순방향**

## 계산기 사용 방법

1. **공급 전압 입력**: 마이크로컨트롤러 GPIO는 3.3V, USB는 5V, 자동차는 12V.
2. **LED 순방향 전압 입력**: 데이터시트의 V_f 참조. 빨강/노랑 ≈ 2.0V, 파랑/흰색/초록 ≈ 3.0V–3.4V.
3. **목표 전류(mA) 입력**: 표준 LED는 보통 20mA로 동작합니다.
4. **직렬 LED 수 입력**: 계산기가 단일 LED 순방향 전압에 이 수를 곱합니다.
5. **결과 확인**: 정확한 저항값, E12 표준값, 소비 전력, 실제 LED 전류, 색상 코드.

## 계산 예시

### 예시 1 — 5V 빨간색 LED

빨간 LED(V_f=2.0V, I=20mA), 5V 전원.

- 저항 양단 전압 = 5.0 − 2.0 = 3.0V
- R = 3.0 / 0.020 = **150Ω** (E12값: 150Ω)
- P = 3.0 × 0.020 = **0.060W**, 1/4W 저항 충분
- 색상 코드: 갈색/초록/갈색/금색

### 예시 2 — 12V 흰색 LED

흰색 LED(V_f=3.2V, I=20mA), 12V 전원.

- R = (12.0 − 3.2) / 0.020 = **440Ω** → E12값: 470Ω
- P = 8.8 × 0.020 = **0.176W**, 1/2W 저항 사용
- 470Ω 실제 전류: 8.8 / 470 = **18.7mA**

### 예시 3 — 12V 파란색 LED 3개 직렬

각 V_f=3.2V, 목표 전류 20mA, 12V 전원.

- 총 순방향 전압 = 9.6V; 저항 양단 전압 = 2.4V
- R = 2.4 / 0.020 = **120Ω** (E12값: 120Ω)
- P = 0.048W, 1/4W 저항 충분

## 자주 묻는 질문

### LED에 전류 제한 저항이 필요한 이유는?
LED는 비선형 소자로 전류가 전압에 따라 지수적으로 증가합니다. 저항이 없으면 전압의 미세한 상승만으로도 밀리초 안에 LED가 파손됩니다.

### E12 저항 시리즈란?
EIA가 정의한 10배 구간당 12개 우선값의 표준: 10, 12, 15, 18, 22, 27, 33, 39, 47, 56, 68, 82 (및 그 10진 배수).

### 왜 더 높은 표준값으로 올림하나요?
올림하면 저항이 커져 LED 전류가 목표보다 약간 낮아집니다. 이것이 안전한 방향입니다.

### 저항 정격 전력은 어떻게 선택하나요?
계산된 소비 전력의 최소 2배 이상. 0.08W이면 0.25W 저항을 사용하세요.
