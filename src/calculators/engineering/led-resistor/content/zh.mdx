# LED限流电阻计算器

## 什么是LED限流电阻？

LED（发光二极管）是一种在电流通过时发光的半导体器件。与普通电阻或白炽灯不同，LED没有内置的电流限制机制。因此，如果没有外部限流电阻，即使电源电压轻微升高，也可能引发破坏性的大电流冲击，永久损坏LED。

串联的限流电阻是最简单、最可靠的解决方案。它将多余的电压（电源电压与LED正向电压之差）转化为热量，从而将LED电流保持在安全可预测的水平。

基本公式直接来自欧姆定律：

**R = (V供电 − V正向 × n) / I正向**

其中：
- **R** = 所需电阻值（Ω）
- **V供电** = 直流电源电压（如5V、12V）
- **V正向** = 单个LED的正向电压（按颜色典型值为1.8V–3.6V）
- **n** = 串联LED数量
- **I正向** = 目标LED电流（安培，如20mA为0.020A）

电阻功耗：**P = (V供电 − V正向 × n) × I正向**

## 如何使用计算器

1. **输入电源电压**：电源的直流电压，如微控制器GPIO的3.3V、USB的5V或汽车应用的12V。
2. **输入LED正向电压**：参考LED数据手册的V_f值。常见值：红/黄约2.0V；蓝/白/绿约3.0V–3.4V。
3. **输入目标电流（mA）**：标准LED通常工作在20mA。
4. **输入串联LED数量**：计算器将单个LED正向电压乘以此数量。
5. **读取结果**：精确电阻值、E12标准值、功耗、实际LED电流和色环代码。

## 计算示例

### 示例一 — 5V红色LED

红色LED（V_f=2.0V，I=20mA），5V电源。

- 电阻两端电压 = 5.0 − 2.0 = 3.0V
- R = 3.0 / 0.020 = **150Ω**（E12值：150Ω）
- P = 3.0 × 0.020 = **0.060W**，使用1/4W电阻
- 色环：棕/绿/棕/金

### 示例二 — 12V白色LED

白色LED（V_f=3.2V，I=20mA），12V电源。

- R = (12.0 − 3.2) / 0.020 = **440Ω** → E12标准值：470Ω
- P = 8.8 × 0.020 = **0.176W**，使用1/2W电阻
- 470Ω实际电流：8.8 / 470 = **18.7mA**

### 示例三 — 12V三串联蓝色LED

每个V_f=3.2V，目标电流20mA，12V电源。

- 总正向电压 = 9.6V；电阻两端电压 = 2.4V
- R = 2.4 / 0.020 = **120Ω**（E12值：120Ω）
- P = 0.048W，1/4W电阻足够

## 常见问题

### LED为什么需要限流电阻？
LED是非线性器件，其电流随电压指数增长。没有电阻时，电压的微小升高会在毫秒内产生大电流，永久损坏LED。

### E12电阻系列是什么？
E12是EIA定义的每十倍频程12个优选电阻值的标准：10、12、15、18、22、27、33、39、47、56、68、82（及其十进位倍数）。

### 为何向上取整到最近的标准值？
向上取整得到更大的电阻，电流略低于目标值，这是安全方向——LED保持在额定电流范围内。

### 应选择多大功率的电阻？
始终选择至少两倍于计算功耗的额定功率。如计算得0.08W，则使用0.25W电阻。
