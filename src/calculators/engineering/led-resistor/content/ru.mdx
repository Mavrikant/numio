# Калькулятор резистора для светодиода

## Что такое токоограничивающий резистор для светодиода?

Светодиод (LED) — это полупроводниковый компонент, который излучает свет при прохождении через него тока. В отличие от обычного резистора или лампы накаливания, светодиод не имеет встроенного механизма ограничения тока. Без внешнего резистора даже небольшое увеличение напряжения питания может привести к разрушительному скачку тока, навсегда повреждающему светодиод.

Последовательный токоограничивающий резистор — самое простое и надёжное решение. Он поглощает избыточное напряжение (разницу между напряжением питания и прямым напряжением светодиода), преобразуя его в тепло, и тем самым удерживает ток светодиода на безопасном уровне.

Основная формула непосредственно следует из закона Ома:

**R = (V_пит − V_пр × n) / I_пр**

Где:
- **R** = требуемое сопротивление в омах (Ом)
- **V_пит** = постоянное напряжение питания (например, 5В, 12В)
- **V_пр** = прямое напряжение одного светодиода в вольтах (обычно 1,8В–3,6В в зависимости от цвета)
- **n** = количество светодиодов в последовательной цепи
- **I_пр** = желаемый ток светодиода в амперах (например, 0,020А для 20мА)

Мощность, рассеиваемая резистором:

**P = (V_пит − V_пр × n) × I_пр**

## Как пользоваться калькулятором

1. **Введите напряжение питания**: Постоянное напряжение источника питания — 3,3В для GPIO микроконтроллера, 5В для USB, 12В для автомобильных схем.
2. **Введите прямое напряжение светодиода**: Смотрите техническую документацию. Типичные значения: красный/жёлтый ≈ 2,0В; синий/белый/зелёный ≈ 3,0В–3,4В.
3. **Введите желаемый ток в мА**: Стандартные светодиоды работают при 20мА.
4. **Введите количество светодиодов в цепи**: Калькулятор умножит прямое напряжение одного светодиода на это число.
5. **Прочтите результаты**: Точное сопротивление, стандартное значение ряда E12, рассеиваемая мощность, реальный ток светодиода и цветовой код резистора.

## Примеры расчёта

### Пример 1 — Светодиод на 5В

Красный светодиод (V_f = 2,0В, I = 20мА) от источника 5В.

- Напряжение на резисторе = 5,0 − 2,0 = 3,0В
- R = 3,0 / 0,020 = **150 Ом** (значение E12: 150 Ом)
- P = 3,0 × 0,020 = **0,060 Вт** — подойдёт резистор 0,25 Вт
- Цветовой код: коричневый / зелёный / коричневый / золотой

### Пример 2 — Белый светодиод на 12В

Белый светодиод (V_f = 3,2В, I = 20мА) от источника 12В.

- R = (12,0 − 3,2) / 0,020 = **440 Ом** → значение E12: 470 Ом
- P = 8,8 × 0,020 = **0,176 Вт** — использовать резистор 0,5 Вт
- Реальный ток с резистором 470 Ом: 8,8 / 470 = **18,7 мА**

### Пример 3 — Три синих светодиода последовательно на 12В

V_f каждого = 3,2В, целевой ток = 20мА, питание = 12В.

- Суммарное прямое напряжение = 9,6В; напряжение на резисторе = 2,4В
- R = 2,4 / 0,020 = **120 Ом** (значение E12: 120 Ом)
- P = 0,048 Вт — резистор 0,25 Вт достаточен

## Часто задаваемые вопросы

### Почему светодиодам нужен токоограничивающий резистор?
Светодиоды — нелинейные приборы: их ток растёт экспоненциально с напряжением. Без резистора небольшой рост напряжения уничтожает светодиод за миллисекунды.

### Что такое ряд резисторов E12?
E12 — стандарт EIA с 12 предпочтительными значениями сопротивлений на декаду: 10, 12, 15, 18, 22, 27, 33, 39, 47, 56, 68, 82 (и их десятичные кратные).

### Почему нужно округлять вверх до ближайшего стандартного значения?
Округление вверх даёт большее сопротивление, слегка снижая ток светодиода. Это безопасное направление: светодиод остаётся в пределах номинального тока.

### Какую мощность резистора выбрать?
Всегда не менее чем вдвое больше рассчитанной рассеиваемой мощности. При 0,08 Вт — используйте резистор 0,25 Вт.

### Можно ли обойтись без резистора при использовании источника постоянного тока?
Да. Источники постоянного тока регулируют ток напрямую и делают резистор излишним. Однако для простых схем с фиксированным напряжением резистор остаётся наиболее экономичным решением.
