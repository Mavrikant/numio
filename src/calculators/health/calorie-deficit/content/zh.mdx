# 热量缺口计算器

想知道减掉X千克需要多长时间？本计算器提供有科学依据的估算。输入您的当前体重、目标体重和每日热量缺口。

## 什么是热量缺口？

热量缺口是指您摄入的热量少于身体消耗的热量。此时，身体会调用储备能量——主要是体脂——来弥补差额。随着时间推移，这将产生可量化的体重减少。

**没有缺口 = 没有减重。** 维持热量使体重保持稳定。

## 7700千卡/千克规则

本计算器使用临床上确立的规则：

**1千克体脂 ≈ 7700千卡的储存能量**

要减掉1千克，需要累计产生约**7700千卡**的总热量缺口：

- 每日500千卡缺口 → 每周3500千卡 → 约0.45千克/周
- 每日1000千卡缺口 → 每周7000千卡 → 约0.9千克/周

## 计算器工作原理

1. **需减体重**：当前体重 − 目标体重
2. **所需总热量**：需减体重 × 7700
3. **每周缺口**：每日缺口 × 7
4. **达到目标所需周数**：总热量 ÷ 每周缺口（向上取整）
5. **达到目标所需月数**：周数 ÷ 4.33

## 计算示例

**示例1 — 适度缺口：**
- 体重：80千克 → 目标：70千克，每日500千卡
- 10千克 × 7700 = 77000千卡 → 22周 → 约5.1个月

**示例2 — 较大缺口：**
- 体重：90千克 → 目标：75千克，每日1000千卡
- 15千克 × 7700 = 115500千卡 → 17周 → 约3.9个月

## 安全缺口大小

| 每日缺口 | 每周减重 | 评估 |
|---|---|---|
| 100–200千卡 | 约0.1千克 | 极慢但高度可持续 |
| 300–500千卡 | 0.3–0.45千克 | 适合大多数人 |
| 500–750千卡 | 0.45–0.65千克 | 适中；注意饥饿感 |
| 750–1000千卡 | 0.65–0.9千克 | 较大；有肌肉流失风险 |
| 超过1000千卡 | 超过0.9千克 | 极大；建议在医疗监督下进行 |

## 为什么减重不是线性的？

计算器提供的是最乐观估算。现实中减重很少是一条直线：

1. **代谢适应**：随着体重下降，身体消耗的热量减少。
2. **水分潴留**：饮食变化、荷尔蒙波动和盐分摄入影响体重秤读数。
3. **平台期**：持续数周缺口后，身体适应，减重速度放缓。
4. **肌肉与脂肪**：没有足够蛋白质和力量训练的大缺口会导致肌肉流失。

## 常见问题

**为什么不能尽量少吃？**
极低热量饮食会导致肌肉流失、营养缺乏和荷尔蒙紊乱。减掉的重量主要是水分和肌肉，往往很快就会反弹。

**如何知道我的每日缺口？**
用BMR计算器和活动系数估算TDEE，然后减去实际热量摄入。

**为什么计算器不显示具体日期？**
减重是非线性的，取决于日常执行情况、代谢适应和生物变异性。显示具体日期会产生误导。
