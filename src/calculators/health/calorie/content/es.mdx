# Calculadora de Calorías — Necesidades Calóricas y TDEE

Conocer tus necesidades calóricas diarias es la base de cualquier planificación nutricional. Tanto si quieres perder peso, ganar músculo o mantener tu figura, el Gasto Energético Total Diario (TDEE) te proporciona los datos necesarios para tomar decisiones dietéticas informadas.

## ¿Qué es el TDEE?

El TDEE es el total de calorías quemadas en un día, incluyendo las funciones basales y toda la actividad física. El principio es sencillo:
- **Comer igual al TDEE** → mantener el peso
- **Comer por debajo** → perder peso (déficit calórico)
- **Comer por encima** → ganar peso (superávit calórico)

## Cómo se calcula

Este calculador usa la **ecuación Mifflin-St Jeor** (1990):
- **Hombre:** TMB = 10 × peso + 6,25 × altura − 5 × edad + 5
- **Mujer:** TMB = 10 × peso + 6,25 × altura − 5 × edad − 161

TDEE = TMB × factor de actividad:

| Nivel de actividad | Factor |
|---|---|
| Sedentario | × 1,2 |
| Ligeramente activo | × 1,375 |
| Moderadamente activo | × 1,55 |
| Muy activo | × 1,725 |
| Extremadamente activo | × 1,9 |

## Ajustes por objetivo

| Objetivo | Ajuste |
|---|---|
| Mantener | ± 0 kcal |
| Perder | − 500 kcal/día |
| Pérdida rápida | − 1000 kcal/día |
| Ganar | + 500 kcal/día |
| Ganancia rápida | + 1000 kcal/día |

**Mínimo de seguridad:** Para objetivos de pérdida, se aplica un mínimo de 1200 kcal/día (mujeres) y 1500 kcal/día (hombres).
