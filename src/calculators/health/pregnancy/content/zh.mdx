# 预产期计算器

## 什么是预产期计算器？

预产期计算器根据末次月经（LMP）的第一天和平均月经周期长度，估算宝宝的预计出生日期。最广泛使用的方法是**内格莱公式**，由19世纪德国产科医生弗朗茨·卡尔·内格莱提出。

内格莱公式在末次月经第一天的基础上加上280天（40周）来计算预产期，假设标准周期为28天。由于各人周期不同，我们的计算器会根据您实际周期与28天的差值进行调整。

**计算公式：**
- 调整后预产期 = 末次月经日期 + 280天 + （周期长度 − 28）天

例如，若您的周期为32天，计算器会在标准280天基础上加4天，即预产期在末次月经后284天。

孕周——即怀孕进展情况——从末次月经第一天算起，而非受孕日期。这是因为排卵和受孕日期难以精确确定，而末次月经起始日期通常是已知的。

## 如何使用计算器

1. **输入末次月经日期：** 以YYYY-MM-DD格式输入末次月经开始的第一天。
2. **输入周期长度：** 填写您月经周期的平均天数（21至35天）。不确定时默认使用28天。
3. **输入今天日期：** 用于计算当前孕周和距预产期的剩余天数。
4. **查看结果：** 计算器显示预产期、当前孕周（整数周和余数天）、所处孕期以及距预产期的剩余天数。

请记住，这只是一个估算值。您的医生可能会通过超声波测量来修正预产期，尤其是在孕早期。

## 计算示例

### 示例一：标准28天周期
- 末次月经日期：2024年1月1日
- 周期长度：28天
- 计算：2024年1月1日 + 280天 = **2024年10月7日**
- 在第10周（末次月经后70天）：孕周 = 10周整，孕早期，剩余210天

### 示例二：较长的32天周期
- 末次月经日期：2024年1月1日
- 周期长度：32天（比标准多4天）
- 调整：+4天
- 预产期 = 2024年1月1日 + 284天 = **2024年10月11日**

### 示例三：较短的24天周期
- 末次月经日期：2024年3月15日
- 周期长度：24天（比标准少4天）
- 调整：−4天
- 预产期 = 2024年3月15日 + 276天 = **2024年12月16日**

## 常见问题

**预产期计算有多准确？**
内格莱公式仅提供估算值。研究表明，只有约4-5%的婴儿恰好在预产期出生，大多数分娩发生在预产期前后两周内。孕早期超声检查是确认孕龄最准确的方法。

**各孕期的时间范围？**
妊娠根据自末次月经起的孕周分为三个孕期：
- **孕早期：** 第1–13周 — 早期发育，器官形成
- **孕中期：** 第14–27周 — 胎儿生长，胎动出现
- **孕晚期：** 第28周至分娩 — 体重快速增加，为分娩做准备

**如果不知道末次月经日期怎么办？**
若不确定具体日期，医生可通过孕早期超声波检查测量胎儿顶臀长来估算孕龄，这是末次月经不明时的标准诊断方法。

**周期长度会影响排卵时间吗？**
会。28天周期通常在第14天排卵，32天周期通常在第18天排卵。根据周期长度进行调整正是为了反映这种排卵时间的差异。

**我能仅凭此计算器做医疗决策吗？**
不能。本工具仅供参考。请务必咨询医生或助产士进行准确的孕周确认和产前保健规划。
