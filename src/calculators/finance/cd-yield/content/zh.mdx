# 存款证书收益计算器

存款证书（CD）是最安全、最可预测的储蓄工具之一。本计算器帮助您了解存款到期时的增值金额、APR与APY的区别，以及提前取款的代价。

## 什么是存款证书？

存款证书允许您以固定利率将固定金额存放一段时间（从一个月到十年）。作为锁定资金的回报，银行支付比普通储蓄账户更高的利率。

存款证书使用复利：赚取的利息定期加入本金，未来利息在更高余额基础上计算。

**未来价值 = P × (1 + APR/n)^(n × t)**

**APY = (1 + APR/n)^n − 1**

### 提前取款罚金

- 短期（12个月以内）：损失90天利息
- 中期（1–2年）：180天利息
- 长期（3年以上）：270–365天利息

## 使用方法

1. 输入本金 — 计划存入的金额。
2. 输入APR — 银行公布的年利率。
3. 设置期限 — 以月为单位。
4. 选择复利频率。
5. 输入提前取款罚金期（可选）。

## 示例

### 示例1：一年期存款证书

10,000美元，APR 5.00%，月复利，12个月。

- APY ≈ **5.116%**
- 到期价值 ≈ **10,511.62美元**
- 利息收益 = **511.62美元**

### 示例2：两年期日复利存款

25,000美元，APR 4.75%，日复利，24个月。

- APY ≈ **4.863%**
- 到期价值 ≈ **27,461美元**
- 利息 ≈ **2,461美元**

### 示例3：提前取款影响

15,000美元，APR 5.5%，18个月期，第9个月取款，罚金期3个月。

- 全期利息：≈1,262美元
- 罚金：≈209美元
- 净收益：≈**622美元**

## 常见问题

**问：我的资金安全吗？**
答：在美国，FDIC成员银行的存款证书受保障，每位存款人每家银行最高25万美元。

**问：梯级存款策略如何运作？**
答：将储蓄分散在不同期限的存款证书中。每张到期后，将收益投入新的长期存款证书。

**问：比较APR还是APY？**
答：始终比较APY，它已包含复利效果，提供真实的横向比较。

**问：无罚金存款证书是什么？**
答：允许在到期前取款而不损失利息，通常利率稍低。

**问：如何最大化收益？**
答：比较多家银行的利率（网络银行通常提供最高利率），使用梯级策略。
