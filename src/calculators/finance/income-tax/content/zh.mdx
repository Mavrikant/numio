# 个人所得税估算器 — 美国与土耳其

本工具估算美国（2024年IRS税率）和土耳其（2024年GVK税率）的联邦个人所得税。适合快速估算，不能替代专业税务顾问。

## 累进税制

两国均采用累进税制：不同收入区间适用不同税率。最高税率仅适用于超过相应门槛的那部分收入。

## 美国 — 2024年联邦税率（单身申报）

| 应税收入 | 税率 |
|---|---|
| 0 – $11,600 | 10% |
| $11,601 – $47,150 | 12% |
| $47,151 – $100,525 | 22% |
| $100,526 – $191,950 | 24% |
| $191,951 – $243,725 | 32% |
| $243,726 – $609,350 | 35% |
| 超过$609,350 | 37% |

2024年标准扣除额：单身$14,600；已婚联合申报$29,200；户主$21,900。

## 土耳其 — 2024年GVK税率

| 应税收入（里拉） | 税率 |
|---|---|
| 0 – 110,000 | 15% |
| 110,001 – 230,000 | 20% |
| 230,001 – 870,000 | 27% |
| 870,001 – 3,000,000 | 35% |
| 超过3,000,000 | 40% |

## 重要限制

本计算器仅涵盖**联邦所得税**。不包括美国州税、FICA税、土耳其SGK缴费（14%）及印花税。如需准确计算，请咨询专业税务人员。
