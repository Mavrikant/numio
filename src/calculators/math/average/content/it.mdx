# Calcolatore di Media

Questo strumento calcola istantaneamente media, mediana, moda, somma e intervallo di un elenco di numeri.

## Cos'è una media?

In statistica esistono tre misure principali di tendenza centrale:

- **Media (aritmetica)** — somma dei valori divisa per il loro numero.
- **Mediana** — valore centrale in un elenco ordinato.
- **Moda** — valore più frequente.

## Formule e passi di calcolo

### Media aritmetica

**Media = (x₁ + x₂ + … + xₙ) / n**

**Esempio:** \{4, 8, 15, 16, 23, 42\} → Somma = 108, n = 6 → Media = **18**

### Mediana

1. Ordinare i valori in ordine crescente.
2. Se n è **dispari**: il valore centrale.
3. Se n è **pari**: media dei due valori centrali.

**Esempio:** \{4, 8, 15, 16, 23, 42\} → (15 + 16) / 2 = **15,5**

### Moda

Contare le occorrenze di ogni valore; il più frequente è la moda.

### Intervallo

**Intervallo = Massimo − Minimo**

Esempio: \{4, 8, 15, 16, 23, 42\} → 42 − 4 = **38**

## Esempi di calcolo

### Esempio 1 — Voti dell'esame

Voti: 72, 85, 90, 68, 74, 85, 91, 78

- **Media:** 643 / 8 = **80,375**
- **Mediana:** Ordinati: 68, 72, 74, 78, 85, 85, 90, 91 → (78 + 85) / 2 = **81,5**
- **Moda:** 85 (due volte) → **85**
- **Intervallo:** 91 − 68 = **23**

### Esempio 2 — Effetto di un valore anomalo

Prezzi delle case (migliaia): 250, 270, 265, 280, 260, 1100

- **Media:** ≈ 404 (distorta dal valore estremo)
- **Mediana:** (265 + 270) / 2 = **267,5** (più rappresentativa)

## Errori frequenti

1. **Usare la media con dati asimmetrici.** Per redditi o prezzi, la mediana è spesso più affidabile.
2. **Confondere la posizione della mediana con il suo valore.** Per \{1, 2, 3, 4\}, la mediana è 2,5, non 3.
3. **Assumere che esista sempre una moda.** Se tutti i valori sono unici, non c'è moda.

## Domande frequenti

**Quando usare la mediana invece della media?**
Usa la mediana quando i dati presentano valori anomali o una distribuzione asimmetrica.

**Che cos'è l'intervallo?**
L'intervallo è la differenza tra il valore massimo e il minimo. Misura la dispersione dei dati.
