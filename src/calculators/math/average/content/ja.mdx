この計算機は、数値リストの平均、中央値、最頻値、合計、範囲を即座に計算します。

## 平均とは？

統計学における中心傾向の3つの主要な指標：

- **平均（算術平均）** — すべての値の合計を個数で割った値。
- **中央値** — 並べ替えたリストの中央の値。
- **最頻値** — 最も多く現れる値。

## 公式と計算手順

### 算術平均

**平均 = (x₁ + x₂ + … + xₙ) / n**

**例：** \{4, 8, 15, 16, 23, 42\} → 合計 = 108、n = 6 → 平均 = **18**

### 中央値

1. 値を昇順に並べ替える。
2. n が**奇数**の場合：中央の値。
3. n が**偶数**の場合：中央2つの値の平均。

**例：** \{4, 8, 15, 16, 23, 42\} → (15 + 16) / 2 = **15.5**

### 最頻値

各値の出現回数を数え、最も多いものが最頻値。

### 範囲

**範囲 = 最大値 − 最小値**

例：\{4, 8, 15, 16, 23, 42\} → 42 − 4 = **38**

## 計算例

### 例1 — 試験の点数

点数：72、85、90、68、74、85、91、78

- **平均：** 643 / 8 = **80.375**
- **中央値：** 並べ替え後：68、72、74、78、85、85、90、91 → (78 + 85) / 2 = **81.5**
- **最頻値：** 85（2回登場）→ **85**
- **範囲：** 91 − 68 = **23**

### 例2 — 外れ値の影響

住宅価格（万円）：250、270、265、280、260、1100

- **平均：** ≈ 404（外れ値で引き上げられる）
- **中央値：** (265 + 270) / 2 = **267.5**（より代表的）

## よくある誤り

1. **歪んだデータに平均を使う。** 収入や価格には中央値の方が適切なことが多い。
2. **中央値の位置と値を混同する。** \{1, 2, 3, 4\}の中央値は2.5であり3ではない。
3. **最頻値が常に存在すると思い込む。** すべての値が1回しか現れない場合、最頻値は存在しない。

## よくある質問

**中央値はいつ使うべきですか？**
外れ値があるデータや偏った分布のとき、中央値が「典型的な値」をより正確に表します。

**範囲とは何ですか？**
範囲は最大値と最小値の差で、データのばらつきを示します。
